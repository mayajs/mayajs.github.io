We are trying to make MayaJS as simple as possible for every type developer. This page will serve as your very guide to seamlessly install and setup Maya JS in your machine. Follow these easy steps to get started.

Installation

To start, open your

  • CLI terminal by pressing Windows + R and type cmd or search command prompt in start menu.
  • Then, type npm i @mayajs/cli -g to install MayaJS globally in your local machine.

Quick Start

Next is to

  • Run maya new <name-of-project> e.g.my-new-app.
  • Run cd <name-of-project> to go inside the your project folder.
  • Run maya serve or maya s to start your MayaJS project.
