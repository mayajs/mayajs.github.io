Introduction to the MayaJS DOCS

A simple Node.js Framework for creating REST API server-side-applications. MayaJS is designed for simplicity and ease of use for beginners.

These following documentations will lead you to be familiar and learn to use the MayaJS framework. From the very beginning to finish, these documentations will serve as your guide in working to your project. From first simple app to projects that each individual can use for enterprises. Tutorials and examples are also provided as your stepping stone to achieve your dream project.
